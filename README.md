# homebrew-tap
## Homebrew packages
add a tap.

	$ brew tap htrb/tap

### Ngraph-GTK Launcher
install Ngraph-GTK with launcer App.

	$ brew install --cask ngraph-gtk-launcher
 
### Ngraph-GTK
install Ngraph-GTK.

	$ brew install ngraph-gtk

### Ngraph-GTK (GTK4 version)
install Ngraph-GTK GTK 4 version.

	$ brew install --HEAD --with-gtk4 ngraph-gtk
